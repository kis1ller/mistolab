# МістоLab — Рейтинг якості життя міст України

Інтерактивний дашборд для порівняння українських міст за інтегральним індексом якості життя. Дані — відкриті джерела, живі API, оновлюваний JSON-файл.

## Що робить сайт

- Рейтинг 51 міста за 4 кластерами та 17 показниками
- Живі дані: погода, AQI (якість повітря), курс USD/EUR від НБУ
- Сторінка кожного міста: радарна діаграма, герб, субметрики
- Темна/світла тема, повністю адаптивний

## Оцінка міст

| Кластер                       | Вага | Джерела                           |
| ----------------------------- | ---- | --------------------------------- |
| Економіка та добробут         | 30%  | Work.ua, ProZorro                 |
| Прозорість та управління      | 20%  | TI Ukraine 2024                   |
| Екологія та здоров'я          | 20%  | Open-Meteo AQI (live)             |
| Інфраструктура та мобільність | 30%  | CityBus API, "Місто без бар'єрів" |

## Структура проекту

```
data/cities.json        ← головний файл даних (редагувати тут)
services/
  openDataService.ts    ← live API: Open-Meteo, НБУ, Wikipedia
  cityAnalysis.ts       ← статичний аналіз міста
components/
  CityDetail.tsx        ← сторінка міста
  RankingTable.tsx      ← таблиця рейтингу
```

## Як оновити дані

Відредагуй `data/cities.json` — це звичайний JSON-масив з усіма містами. Кожне місто має поля `metrics`, `subMetrics`, `highlights` тощо. Після редагування:

```bash
npm run build
```

## Запуск локально

```bash
npm install
npm run dev        # dev-сервер на localhost:3000
npm run build      # збірка в dist/
npm run preview    # перегляд dist/ локально
```
